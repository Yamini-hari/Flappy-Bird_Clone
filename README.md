# Flappy-Bird-Clone

Flappy Bird is a side-scrolling mobile game with 2D retro style graphics. The objective of the
game is to direct a flying bird. The bird is flying towards right direction constantly while it has to avoid the pipes that blocks its way.Controlling the bird while avoiding and navigating as many of the approaching tubes as you can is the objective of the game. Until the bird eventually lands on one of the tubes, the ground, or the ceiling, the game never ends.The design includes- XML(Extensible markup language) as frontend and with Java programming language as the backend .
